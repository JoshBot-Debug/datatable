## @jjmyers/datatable

### Check out the [DEMO WEBSITE](https://joshbot-debug.github.io/datatable)

A server sided datatable equiped with every commonly used filter.

## Quick Features
- Sorting
- Multi Sorting
- Number filter
- Text filter
- Boolean filter
- Set filter
- Pagination
- Toggleable columns
- A Seperate controller from component, to allow customization.
